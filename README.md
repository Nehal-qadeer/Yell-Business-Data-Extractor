# 🔒 Business Lead Extraction Pipeline

> **Note:** This repository is a professional portfolio piece showcasing a custom solution developed for a commercial client. 

### 📋 Project Case Study
**The Challenge:** The client required a scalable way to gather business listings across multiple regions for market research. Manual entry was inefficient and prone to errors.

**The Solution:** I developed a multi-stage automation tool using **Python** and **Selenium**. 
* **Link Harvesting:** Iterates through town lists to collect business profile URLs.
* **Data Extraction:** Scrapes specific details (Phone, Address, Ratings) from dynamic pages.
* **Data Structuring:** Automatically cleans and saves the output to CSV for direct use in CRM systems.

**Technical Stack:** * Python (Selenium, CSV)
* Jupyter Notebooks (for iterative data cleaning)
